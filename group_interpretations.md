# Which Cafés Are Best for Different User Types?

## Overview

Different behavioral groups value different café characteristics. This analysis identifies which cafés are best suited for each user type based on group-specific rankings.

**Key insight:** The same café can be top-ranked for one group but not another. Behavioral context matters.
